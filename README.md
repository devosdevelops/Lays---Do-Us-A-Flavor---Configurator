# Three.js Vite Frontend Scaffold

This is a starter structure for a Vite-based Three.js frontend project using vanilla JavaScript and ES modules.

## Structure
- `src/` — Main source code
  - `main.js` — Entry point
  - `scene/` — Scene setup modules
  - `ui/` — UI controls
  - `services/` — API service placeholder
  - `styles/` — CSS
- `index.html` — Canvas mount only
- `.env.example` — Example env file
- `.gitignore` — Ignore node_modules, dist, .env, OS files
- `package.json` — Project metadata

## Getting Started
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## Notes
- Three.js is installed and imported, but not used yet
- No model assets or API logic included
